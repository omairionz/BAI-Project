# UVA CS Course Descriptions and Student Tips

## Introduction to Programming

### CS 1110 / CS 1111 / CS 1112 / CS 1113 - Introduction to Programming (3 credits)
**Prerequisites:** None

Introduces students to programming and problem solving using Python. Topics include variables, conditionals, loops, functions, lists, dictionaries, file I/O, and basic object-oriented programming. CS 1110 is the standard lecture section. CS 1111 is for students with some prior programming experience. CS 1112 includes lab sections. CS 1113 is a special section.

**Student Tips:**
- Go to office hours early and often — the department has many TAs
- Start assignments early; they take longer than they look
- If you have prior experience, consider CS 1111 or the place-out test
- This course is required for both BACS and BSCS (or can be satisfied with AP CS A with score 4 or 5)

## Foundation Courses

### CS 2100 - Data Structures and Algorithms 1 (4 credits)
**Prerequisites:** CS 111X

Covers fundamental data structures including lists, stacks, queues, trees, heaps, and hash tables. Introduces algorithm analysis using Big-O notation. Topics include recursion, sorting algorithms (merge sort, quicksort), tree traversals, and graph basics. Programming in Java.

**Difficulty:** Medium
**Student Tips:**
- This is a major jump from CS 1110 — expect it to be significantly harder
- Java is introduced here; give yourself time to learn the new language
- Must earn a C+ or higher to declare both the BACS and BSCS
- Understanding recursion thoroughly is critical — it appears everywhere in later courses
- Lab sections are part of the course — attend them

### CS 2120 - Discrete Mathematics and Theory 1 (3 credits)
**Prerequisites:** CS 111X

Introduces the mathematical foundations of computer science. Topics include logic, proofs, sets, functions, relations, induction, and basic combinatorics. This course develops the formal reasoning skills needed for upper-level theory courses.

**Difficulty:** Medium
**Student Tips:**
- Very different from typical programming courses — more like math class
- Learning to write proofs takes practice; start assignments early
- Understanding logic and induction well pays off hugely in CS 3120
- Office hours are very helpful for proof-writing feedback

### CS 2130 - Computer Systems and Organization 1 (4 credits)
**Prerequisites:** CS 111X

Covers how computers work at a low level. Topics include binary representation, Boolean logic, digital circuits, assembly language, C programming, memory management, and the relationship between hardware and software.

**Difficulty:** Medium
**Student Tips:**
- C programming is introduced here — a very different style from Python/Java
- Pointer and memory concepts can be tricky; start early on assignments
- This course is critical — it's a prerequisite for CS 3130 which many find the hardest course
- Lab assignments test your understanding of actual machine behavior

### CS 3100 - Data Structures and Algorithms 2 (3 credits)
**Prerequisites:** CS 2100 and CS 2120

Builds on CS 2100 with more advanced algorithms. Topics include shortest path algorithms (Dijkstra's, Bellman-Ford), dynamic programming, greedy algorithms, divide-and-conquer, backtracking, amortized analysis, and reductions. Analysis techniques include asymptotic worst case and expected time.

**Difficulty:** Hard
**Student Tips:**
- Many students consider this one of the hardest CS courses at UVA
- Requires strong understanding of both data structures (from CS 2100) and proofs (from CS 2120)
- Dynamic programming problems require a lot of practice — do many examples
- Avoid taking this in the same semester as CS 3130 if possible

### CS 3120 - Discrete Mathematics and Theory 2 (3 credits)
**Prerequisites:** CS 3100

Covers computability and complexity theory. Topics include finite automata, regular languages, context-free grammars, pushdown automata, Turing machines, decidability, and P vs NP. This course explores the fundamental limits of what computers can and cannot compute.

**Difficulty:** Hard
**Student Tips:**
- Very theoretical — the most math-heavy CS course most students take
- Build on proof skills from CS 2120; weak proof skills will make this very hard
- Turing machines and reductions are the most challenging topics — start studying them early
- This course answers deep questions about what computers can and can't do

### CS 3130 - Computer Systems and Organization 2 (4 credits)
**Prerequisites:** CS 2130 and CS 2100

Goes deeper into computer systems. Topics include operating systems concepts, processes, threads, concurrency, memory hierarchy, caching, virtual memory, and system-level programming in C.

**Difficulty:** Hard
**Student Tips:**
- Widely considered one of the hardest courses in the CS curriculum
- Concurrency bugs (race conditions, deadlocks) are notoriously tricky to debug
- C programming skills from CS 2130 are essential — be sure you're solid on pointers and memory
- Assignments are complex and time-consuming — start them very early
- Office hours are crowded near deadlines; go early in the week

### CS 3140 - Software Development Essentials (3 credits)
**Prerequisites:** CS 2100

Covers software engineering practices and tools used in real development. Topics include version control (Git), testing, refactoring, design patterns, and agile development practices. Includes team-based projects.

**Difficulty:** Medium
**Student Tips:**
- More practical than theoretical — good preparation for internships and jobs
- Learn Git well; it's used in every professional software job
- Team projects require coordination — establish clear communication with teammates early
- Lighter on theory than other foundation courses; more focus on good coding habits

## Upper-Level Required Courses (BSCS only)

### CS 3240 - Software Engineering (3 credits)
**Prerequisites:** CS 3140
**Required for:** BSCS only (not required for BACS)

A project-based course in software engineering. Students work in teams to design, build, and deploy a real software application from scratch. Topics include requirements gathering, system design, agile methodology, testing, and deployment. Often involves a Django or similar web framework project.

**Difficulty:** Medium (but time-intensive)
**Student Tips:**
- The team project is significant — communication and coordination with teammates is critical
- Start the project early each sprint; do not wait until the last week
- Git workflow and code reviews are used extensively
- Many students say this course taught them the most about real software development
- Choose your team carefully if given the option

## CS Capstone Options (BSCS only)

### CS 4980 - Capstone Research (1-3 credits)
Work one-on-one with a faculty member on an original research project. Must complete 3 total credits. Results in a technical report that satisfies the SEAS Senior Thesis requirement. Good option if you are interested in graduate school or research.

### CS 4970 / CS 4971 - Capstone Practicum I & II (3 credits each)
A year-long team project course with a real client (company or nonprofit). CS 4970 in the fall, CS 4971 in the spring. Involves building a real system for an actual customer. Good option if you prefer applied, team-based work.

### CS 4991 - Capstone Technical Report (0 credits)
Must be taken alongside a 3-credit CS 4000-level course not used for any other requirement. Results in a technical report. Good option if you want to take a specific upper-level CS course while also completing the capstone.

## Common CS Electives (3000-level and above)

These are popular CS elective choices. All must have the CS mnemonic and cannot duplicate required/foundation courses.

### CS 4102 - Algorithms (3 credits)
Advanced algorithm design and analysis. Very useful for technical interview preparation. Covers dynamic programming, graph algorithms, and NP-completeness in depth.

### CS 4501 - Topics in CS (3 credits, varies by semester)
Special topics courses offered on subjects like Machine Learning, Natural Language Processing, Computer Vision, Cybersecurity, and more. Topics rotate each semester — check the course catalog.

### CS 4640 - Programming Languages (3 credits)
Covers the theory and design of programming languages. Topics include type systems, functional programming, and language semantics.

### CS 4457 - Computer Networks (3 credits)
Covers how the internet works: protocols, TCP/IP, routing, DNS, HTTP, and network security basics.

### CS 4750 - Database Systems (3 credits)
Covers relational databases, SQL, query optimization, and transaction management.

### CS 4810 - Computer Graphics (3 credits)
Covers rendering, 3D geometry, shaders, and graphics pipelines. Math-intensive but visually rewarding.

## Applied Mathematics Courses (BSCS requirement)

### APMA 3100 - Probability (3 credits)
Required for all BSCS students. Covers probability theory, random variables, distributions, and expected values. Very useful for machine learning and algorithms.

### APMA 3080 - Linear Algebra (3 credits)
Covers vectors, matrices, eigenvalues, and linear transformations. Highly recommended for students interested in machine learning or computer graphics.

### APMA 3120 - Statistics (3 credits)
Covers statistical inference, hypothesis testing, and regression. Note: APMA 3120 and APMA 3150 cannot both count toward the BSCS degree.

### APMA 3150 - From Data to Knowledge (3 credits)
Applied statistics with a data science focus. Note: APMA 3120 and APMA 3150 cannot both count toward the BSCS degree.

### APMA 2130 - Ordinary Differential Equations (4 credits)
Covers differential equations. Less directly relevant to CS than linear algebra or statistics, but counts toward the BSCS APMA requirement.
